Spin

The operators for measuring spin (for spin \(=1/2\) particles such as electrons) in the \(x\), \(y\), and \(z\) directions are defined as: \[ S_x=\frac{\hbar}{2}\begin{pmatrix}0&1\\1&0\end{pmatrix}, S_y=\frac{\hbar}{2}\begin{pmatrix}0&-i\\i&0\end{pmatrix}, S_z=\frac{\hbar}{2}\begin{pmatrix}1&0\\0&-1\end{pmatrix} \] where the possible measured values are the eigenvalues of these operators, \(\pm\frac{\hbar}{2}\), and the corresponding eigenvectors are: \[ \mid S_z,+\rangle = \mid +\rangle = \begin{pmatrix}1\\0\end{pmatrix}, \mid S_z,-\rangle = \mid -\rangle = \begin{pmatrix}0\\1\end{pmatrix} \] \[ \mid S_x,\pm\rangle = \frac{\mid +\rangle\pm\mid -\rangle}{\sqrt2}, \mid S_y,\pm\rangle = \frac{\mid +\rangle\pm i\mid -\rangle}{\sqrt2} \]

For discussion on how spin is experimentally observed, see the Stern-Gerlach experiment. For discussion on measuring spin, see Spin Measurements. The derivation for the quantities defined above is found below.

Derivation

We begin by simply defining spin states in the \(z\) direction as: \[\mid S_z,+\rangle = \mid +\rangle = \begin{pmatrix}1\\0\end{pmatrix}, \mid S_z,-\rangle = \mid -\rangle = \begin{pmatrix}0\\1\end{pmatrix}\] which is done by convention. You could start by defining states along some other axis in this exact way, but you would end up with a set of definitions in terms of that other axis. We also start by saying that the eigenvalues are \(\pm\frac{\hbar}{2}\), which we will just say we know as a fact because Dr. Argenti said so and I trust him implicitly...

As we know from our polarized light analogy and our quantum postulates, observing the spin along the \(z\) axis (which is the action of operator \(S_z\)) acts as a projector of the initial state into the eigenstates of \(S_z\).

So, if we apply \(S_z\) on a system in state \(\mid +\rangle\), we should observe a value of \(\frac{\hbar}{2}\), and if we apply \(S_z\) on a system in state \(\mid -\rangle\), we should observe a value of \(-\frac{\hbar}{2}\). This means that we can simply define \(S_z\) as: \[ \frac{\hbar}{2} \mid +\rangle \langle+\mid - \frac{\hbar}{2} \mid -\rangle \langle-\mid = \frac{\hbar}{2}\begin{pmatrix}1&0\\0&0\end{pmatrix}-\frac{\hbar}{2} \begin{pmatrix}0&0\\0&1\end{pmatrix} = \frac{\hbar}{2}\begin{pmatrix}1&0\\0&-1\end{pmatrix} \] We can think of the elements of this matrix as \(\langle+\mid S_z\mid+\rangle\), \(\langle-\mid S_z\mid-\rangle\), and so on, where \(\langle+\mid +\rangle=\langle-\mid -\rangle=1\) and \(\langle+\mid -\rangle=\langle-\mid +\rangle=0\).
From here, we define the spin states and operators in the \(x\) and \(y\) directions.

As we know from our polarized light analogy, matter can act like a wave, namely we can write the state of a system as linear combination of states, meaning that \(\mid S_x,+\rangle\) can be written as a linear combination of \(\mid\pm\rangle\) just like light: \[ \mid S_x,+\rangle=\frac{1}{\sqrt2}e^{i\phi_+}\mid+\rangle+\frac{1}{\sqrt2}e^{i\phi_-}\mid-\rangle \tag{1} \]

An aside about phase factors. Our \(e^{i\phi_\pm}\) terms describe the phase factor of each state \(\mid\pm\rangle\). In terms of our light analogy, the amplitude and frequency of the wave do not change, the wave is just shifted so the phase is different.

In terms of quantum mechanics: the phase factor \(e^{i\phi_\pm}\) is a scalar, so \(e^{i\phi_\pm}\mid\pm\rangle\) just scales a ket or bra. This means the introduction of a phase factor does not change the expectation values of a Hermitian operator with respect to whatever new scaled state \(e^{i\phi_\pm}\mid\pm\rangle\) we have. That is, the values of \(\langle\phi\mid A \mid\phi\rangle\) and \(\langle\psi\mid A \mid\psi\rangle\), where \(\mid\psi\rangle=e^{i\theta}\mid\phi\rangle\), are the same.

In even simpler terms, an eigenket modified by a phase factor is still an eigenket of that operator.

Because we can arbitrarily asign phase factors, we can re-define \(\mid S_x,+\rangle\) as \(\mid S_x,+\rangle e^{-i\phi_+}\), which allows us to re-write (1) as: \[ \begin{align} \mid S_x,+\rangle &= \frac{1}{\sqrt2}e^{i\phi_+}\mid+\rangle+\frac{1}{\sqrt2}e^{i\phi_-}\mid-\rangle \\ \mid S_x,+\rangle e^{-i\phi_+} &= \frac{1}{\sqrt2}e^{-i\phi_+}e^{i\phi_+}\mid+\rangle+\frac{1}{\sqrt2}e^{-i\phi_+}e^{i\phi_-}\mid-\rangle \\ &= \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i(\phi_--\phi_+)}\mid-\rangle \\ \end{align} \] We can further re-define \(\mid -\rangle\) as \(e^{i(-\phi_-+\phi_+)}\mid-\rangle\) and say: \[ \begin{align} \mid S_x,+\rangle &= \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i(\phi_--\phi_+)}e^{i(-\phi_-+\phi_+)}\mid-\rangle \\ &= \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}\mid-\rangle \tag{2} \\ \end{align} \] Which is exactly as we defined above. Yay! Next, we can begin to describe \(\mid S_x,-\rangle\) as we did for \(\mid S_x,+\rangle \): \[ \begin{align} \mid S_x,-\rangle &= \frac{1}{\sqrt2}e^{i\theta_+}\mid+\rangle+\frac{1}{\sqrt2}e^{i\theta_-}\mid-\rangle \\ \mid S_x,-\rangle e^{-i\theta_+} &= \frac{1}{\sqrt2}e^{-i\theta_+}e^{i\phi_+}\mid+\rangle+\frac{1}{\sqrt2}e^{-i\theta_+}e^{i\theta_-}\mid-\rangle \\ &= \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i(\theta_--\theta_+)}\mid-\rangle \\ \end{align} \] For simplicity's sake let's also re-define \(\theta_-=\theta_--\theta_+\), so we have: \[ \mid S_x,-\rangle = \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i\theta_-}\mid-\rangle \] How do we find \(\theta_-\)? We know that \(\mid S_x,+\rangle\) and \(\mid S_x,-\rangle\) are orthogonal due to the observations during the Stern-Gerlach experiment, so: \[ \begin{align} \langle S_x, +\mid S_x,-\rangle &= (\frac{1}{\sqrt2}\langle+\mid+\frac{1}{\sqrt2}\langle -\mid) (\frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i\theta_-}\mid-\rangle) \\ &= (\frac{1}{\sqrt2}\langle+\mid+\frac{1}{\sqrt2}\langle -\mid)(\frac{1}{\sqrt2}\mid+\rangle) + (\frac{1}{\sqrt2}\langle+\mid+\frac{1}{\sqrt2}\langle -\mid)(\frac{1}{\sqrt2}e^{i\theta_-}\mid-\rangle)\\ &= |\frac{1}{\sqrt2}|^2\langle+\mid+\rangle + |\frac{1}{\sqrt2}|^2e^{i\theta_-}\langle-\mid-\rangle \\ &= \frac{1}{2}\langle+\mid+\rangle + \frac{1}{2}e^{i\theta_-}\langle-\mid-\rangle \\ &= \frac{1}{2} + \frac{1}{2}e^{i\theta_-} \\ \end{align} \] Clearly, we need \(e^{i\theta_-}=-1\). According to Euler's formula we can say that \[ \begin{align} e^{i\theta_-} &= cos(\theta_-)+isin(\theta_-) \\ \theta_-=\pi, e^{i\theta_-} &= cos(\pi)+isin(\pi) \\ &= -1+0 \\ &= -1 \end{align} \] So \(\theta_-=\pi\), and we can finally write: \[ \mid S_x,-\rangle = \frac{1}{\sqrt2}\mid+\rangle+\frac{1}{\sqrt2}e^{i\theta_-}\mid-\rangle = \frac{1}{\sqrt2}\mid+\rangle-\frac{1}{\sqrt2}\mid-\rangle \tag{3} \] We then combine (2) and (3) to get the defintion at the top of this page: \[\mid S_x,\pm\rangle = \frac{\mid +\rangle\pm\mid -\rangle}{\sqrt2}\] We find the operator for \(S_x\) as we did for \(S_z\): \[\begin{align} S_x &= \frac{\hbar}{2} \mid S_x,+\rangle \langle S_x,+\mid - \frac{\hbar}{2} \mid S_x,-\rangle \langle S_x,-\mid \\ &= \frac{\hbar}{2}(\frac{\mid+\rangle+\mid-\rangle}{\sqrt2})(\frac{\langle+\mid+\langle-\mid}{\sqrt2}) - \frac{\hbar}{2}(\frac{\mid+\rangle-\mid-\rangle}{\sqrt2})(\frac{\langle+\mid-\langle-\mid}{\sqrt2}) \\ &= \frac{\hbar}{4}(\mid+\rangle+\mid-\rangle)(\langle+\mid+\langle-\mid) - \frac{\hbar}{4}(\mid+\rangle-\mid-\rangle)(\langle+\mid-\langle-\mid) \\ &= \frac{\hbar}{4}(\mid+\rangle\langle+\mid + \mid+\rangle\langle-\mid + \mid-\rangle\langle+\mid + \mid-\rangle\langle-\mid - \mid+\rangle\langle+\mid \\ &\qquad\qquad +\mid+\rangle\langle-\mid +\mid-\rangle\langle+\mid -\mid-\rangle\langle-\mid) \\ &= \frac{\hbar}{4}(2\mid+\rangle\langle-\mid + 2\mid-\rangle\langle+\mid) \\ &= \frac{\hbar}{2}(\mid+\rangle\langle-\mid + \mid-\rangle\langle+\mid) \\ \end{align}\] Clearly, we can write the projectors for \(\mid S_x,+\rangle\) and \(\mid S_x,-\rangle\) as matrices, giving us: \[ \mid S_x, +\rangle\langle S_x,+\mid = \frac{\hbar}{2}\begin{pmatrix}0&1\\0&0\end{pmatrix}, \mid S_x, -\rangle\langle S_x,-\mid = \frac{\hbar}{2}\begin{pmatrix}0&0\\1&0\end{pmatrix} \implies S_x=\frac{\hbar}{2}\begin{pmatrix}0&1\\1&0\end{pmatrix} \] Note that the non-zero elements of this matrix are \(\langle+\mid S_x\mid-\rangle=\frac{\hbar}{2}\) and \(\langle-\mid S_x\mid+\rangle=\frac{\hbar}{2}\)...

We finally turn our attention to the states \(\mid S_y,\pm\rangle\). Similar to the previous demonstration, we write: \[\begin{align} \mid S_y,\pm\rangle &= \frac{1}{\sqrt2}e^{i\delta_+}\mid+\rangle\pm\frac{1}{\sqrt2}e^{i\delta_-}\mid-\rangle \\ \end{align}\] and then re-define \(\mid S_y,\pm\rangle\) with a phase factor to remove the phase factor on the \(\mid+\rangle\) terms: \[\begin{align} \mid S_y,\pm\rangle &= \frac{1}{\sqrt2}\mid+\rangle\pm\frac{1}{\sqrt2}e^{i\delta_-}\mid-\rangle \\ \end{align}\] Experimentally, we know that if we observe the spin along one axis, filter atoms so we know that they are all in the same known state, (e.g. state \(\mid S_x,+\rangle\)) then when we observe the spin along a different axis, the beam is split evenly into two (see variations 2 and 3 in the Stern-Gerlach experiment). We can represent this mathematically as: \[ \begin{align} \mid\langle S_x, +\mid S_y, \pm\rangle\mid^2 = \frac{1}{2} \end{align} \] which of course takes advantage of the measurement posulates of quantum mechanics. \[ \begin{align} \mid\langle S_x, +\mid S_y, \pm\rangle\mid^2 &= \frac{1}{2} \\ &= \mid(\frac{\langle+\mid-\langle-\mid}{\sqrt2})(\frac{1}{\sqrt2}\mid+\rangle\pm\frac{1}{\sqrt2}e^{i\delta_-}\mid-\rangle)\mid^2 \\ &= \mid\frac{1}{2} \pm \frac{1}{2}e^{i\delta_-}\mid^2 \\ &= \mid\frac{1}{2}(1 \pm e^{i\delta_-})\mid^2 \\ &= \frac{1}{4} (1 \pm e^{i\delta_-})(1 \pm e^{-i\delta_-}) \\ &= \frac{1}{4} (1 \pm (e^{i\delta_-}+e^{-i\delta_-}) + 1) \\ &= \frac{1}{4} (2 \pm (e^{i\delta_-}+e^{-i\delta_-})) \\ &= \frac{1}{4} (2 \pm (cos(\delta_-)+isin(\delta_-)+cos(\delta_-)-isin(\delta_-))) \\ &= \frac{1}{4} (2 \pm 2cos(\delta_-)) \\ &= \frac{1}{2} \pm \frac{1}{2}cos(\delta_-) \\ \end{align} \] Clearly, we need \(\frac{1}{2}cos(\delta_-)=0\), which means we can pick \(\delta_-=\frac{\pi}{2}\). Plugging this back into our initial representation of \(\mid S_y,\pm\rangle\): \[\begin{align} \mid S_y,\pm\rangle &= \frac{1}{\sqrt2}\mid+\rangle\pm\frac{1}{\sqrt2}e^{i\delta_-}\mid-\rangle \\ &= \frac{1}{\sqrt2}\mid+\rangle\pm\frac{1}{\sqrt2}e^{i\frac{\pi}{2}}\mid-\rangle \\ &= \frac{1}{\sqrt2}\mid+\rangle\pm\frac{1}{\sqrt2}i\mid-\rangle \\ \end{align}\] because \(e^{i\frac{\pi}{2}}=cos(\frac{\pi}{2})+isin(\frac{\pi}{2})=0+i=i\). Now we can write \[\mid S_y,\pm\rangle = \frac{\mid+\rangle\pm i\mid-\rangle}{\sqrt2}\] Note that because we define the eigenvalues and eigenvectors of \(S_x\) (and \(S_z\), since we could have done the above derivation with \(\mid\langle S_z, +\mid S_y, \pm\rangle\mid^2\) instead...) to be real, the eigenvectors of \(S_y\) are imaginary.

The matrix for the operator \(S_y\) then is: \[\begin{align} S_y &= \frac{\hbar}{2} \mid S_y,+\rangle \langle S_y,+\mid - \frac{\hbar}{2} \mid S_y,-\rangle \langle S_y,-\mid \\ &= \frac{\hbar}{2}(\frac{\mid+\rangle+i\mid-\rangle}{\sqrt2})(\frac{\langle+\mid-i\langle-\mid}{\sqrt2}) - \frac{\hbar}{2}(\frac{\mid+\rangle-i\mid-\rangle}{\sqrt2})(\frac{\langle+\mid+i\langle-\mid}{\sqrt2}) \\ &= \frac{\hbar}{4}(\mid+\rangle+i\mid-\rangle)(\langle+\mid-i\langle-\mid) - (\mid+\rangle-i\mid-\rangle)(\langle+\mid+i\langle-\mid) \\ &= \frac{\hbar}{4}(-2i\mid+\rangle\langle-\mid + 2i\mid-\rangle\langle+\mid) \\ &= \frac{\hbar}{2}(-i\mid+\rangle\langle-\mid + i\mid-\rangle\langle+\mid) \\ \end{align}\] Clearly, we can write the projectors for \(\mid S_x,+\rangle\) and \(\mid S_x,-\rangle\) as matrices, giving us: \[ \mid S_y, +\rangle\langle S_y,+\mid = \frac{\hbar}{2}\begin{pmatrix}0&-i\\0&0\end{pmatrix}, \mid S_y, -\rangle\langle S_y,-\mid = \frac{\hbar}{2}\begin{pmatrix}0&0\\i&0\end{pmatrix} \implies S_y=\frac{\hbar}{2}\begin{pmatrix}0&-i\\i&0\end{pmatrix} \] Note that the non-zero elements of this matrix are \(\langle+\mid S_y\mid-\rangle=-i\) and \(\langle-\mid S_y\mid+\rangle=i\)...
If you know anything about the Pauli matrices, we can say: \[ S_x=\frac{\hbar}{2}\sigma_1, S_y=\frac{\hbar}{2}\sigma_2, S_z=\frac{\hbar}{2}\sigma_3 \]
