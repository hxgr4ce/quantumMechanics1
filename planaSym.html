Planar Symmetries of an Equliateral Triangle

Symmetries

A symmetry is a transformation that maps a triangle onto itself (note that the points and edges can be interchanged, as long as the resulting triangle is otherwise identical).

We can form a set of six symmetries for an equilateral triangle using identity, reflection, and rotation transformations.

1. Identity: Make no changes to the original triangle. 2. Reflection: There are three reflections. Hold one corner constant, flip the other two: 3. Rotation: There are two reflections, a clockwise and a counter clockwise rotation of \(120^\circ\).

Proving Group Properties

Say we call the identity transformation \(\mathbb{I}\), the reflection with point \(i\) held constant \(ref_i\), and the rotations \(rot_{cw}\) and \(rot_{ccw}\).

Is the set S = \(\{\mathbb{I}, ref_A, ref_B, ref_C, rot_{cw}, rot_{ccw}\}\) a group? Yes!

  • Associative: \(\forall a, b \in S, (a+b)+c = a+(b+c)\) holds, since the grouping of transformations does not matter as long as their order is preserved.

    E.g. \(rot_{cw} + (ref_A + ref_B) = (rot_{cw} + ref_A) + ref_B\). The transformation \(ref_A + ref_B\) is the same as \(rot_{cw}\):

    and the transformation \(rot_{cw} + ref_A\) is the same as \(ref_C\):

    Thus \[ \begin{align} rot_{cw} + (ref_A + ref_B) & = rot_{cw} + rot_{cw} \\ & = rot_{ccw} \\ (rot_{cw} + ref_A) + ref_B & = ref_C + ref_B \\ & = rot_{ccw} \\ rot_{cw} + (ref_A + ref_B) & = (rot_{cw} + ref_A) + ref_B \end{align} \]

    Coming up with the full table of pairwise transformations and their equivalent single transformations to prove this for all \(s \in S\) is left as an exercise for the reader :)

  • The neutral element in this case is clearly \(\mathbb{I}\), since \(\forall s \in S, \mathbb{I} + s = s+\mathbb{I} = s\)
  • Each symmetry has an inverse, or a corresponding symmetric element:
    • Each \(ref_i\) is the inverse of itself, so that \(ref_i + ref_i = \mathbb{I}\)
    • \(rot_{cw} + rot_{ccw} = \mathbb{I}\), and \(rot_{ccw} + rot_{cw} = \mathbb{I}\)
    • \(\mathbb{I} + \mathbb{I} = \mathbb{I}\)

So S is a group.
