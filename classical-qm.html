The Bridge Between Classical and Quantum Mechanics

Say that we have a point mass moving in one dimension with mass \(m\) and position \(x\):

\(x\in\mathbb{R}\), and since we can measure position, we know there must be an associated observable \(\hat{X}\) which is self-adjoint. Since \(\hat{X}\) is self-adjoint, we know that it has as many eigenvalues as it does possible measured values. \[\hat{X}|x\rangle=x|x\rangle, \sigma(\hat{X})=\mathbb{R}\tag{1}\] This raises two obvious issues:
  1. The spectrum of \(\hat{X}\) is continuous
  2. The spectrum of \(\hat{X}\) is unbounded
This first issue leads us down the following rabbithole: We can no longer say that: \[\sum_x |x\rangle\langle x|=\hat{1}\] because there is no discrete \(x\) to sum over. Instead, we have to say: \[\hat{1}=\int_{-\infty}^{\infty} dx |x\rangle\langle x|\] Clearly, the familiar \(|x\rangle\langle x|\) is no longer a projector, it is a projector density, \(\frac{d\hat{P}}{dt}\). We also run into the following issue: \[\begin{align} |\psi\rangle=\hat{1}|\psi\rangle &= \int_{-\infty}^{\infty} dx |x\rangle\langle x|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx |x\rangle\psi(x) \\ \langle x|\psi\rangle &= \langle x|\hat{1}|\psi\rangle = \langle x|\int_{-\infty}^{\infty} dx' |x'\rangle\langle x'|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx' \langle x|x'\rangle\langle x'|\psi\rangle \\ \end{align}\] (Note that here, we represent \(\langle x|\psi\rangle\) as \(\psi(x)\) which is the position-space wave function.) The only time that each part of this integral is not 0 is when \(x'=x\), so: \[ \langle x|\psi\rangle = \int_{-\infty}^{\infty} dx' \delta(x-x')\langle x'|\psi\rangle \] (Note that this is the Dirac Delta, not the Kroenecker Delta, since the integral of the Dirac Delta is 1. ) This indicates that \(\langle x|x\rangle=\delta(0)=\infty\), so \(|x\rangle\) is not normalizable.

In this situation, \(|x\rangle\) is not a ket that represents the state of quantum system, it is a generalized eigenstate which is used as a mathematical tool only.

The second issue leads us to the following conundrum: The Hellinger-Toeplitz Theorem says that if an operator \(\hat{A}\) is defined on the whole Hilbert space, it is bounded.

As a corollary, if we know in advance that an operator \(\hat{A}]\) is unbounded, we know that it is not defined on the whole Hilbert space.

Some operators have a non-trivial domain.

Take, for example, our position operator \(\hat{X}\): For some state \(|\psi\rangle\), we know that \(\langle\psi|\psi\rangle\lt\infty\). Then: \[ \langle\psi|\hat{1}|\psi\rangle = \langle\psi|\int_{-\infty}^{\infty} dx|x\rangle\langle x|\psi\rangle = \int_{-\infty}^{\infty} dx\langle\psi|x\rangle\langle x|\psi\rangle = \int_{-\infty}^{\infty} dx|\psi(x)|^2 \lt \infty \] Which tells us that \(\psi(x)\in L^2(\mathbb{R})\), or the set of square-integrable functions (functions where \(\int_{-\infty}^{\infty} dx|f(x)|^2 \lt \infty\) is true). This means that we have some functional \(\langle x |\) which has the following action: \[\langle x |:H\to L^2(\mathbb{R}), |\psi\rangle \mapsto \langle x |\psi\rangle\tag{2}\] Based on what we know about linear algebra, we also know that \(\langle x |\psi\rangle\) is the component of the state vector \(|\psi\rangle\) at some position (eigenvalue) \(x\), which corresponds to eigenstate \(|x\rangle\) of \(\hat{X}\) (see definition in (1)).

So, to be able to determine the position of a system in state \(|\psi\rangle\), we first need to project it into the position basis: \(\hat{X}|\psi\rangle\), then check if it exists at a certain position: \(\langle x|\hat{X}|\psi\rangle\).

This will only work if the result of \(\langle x|\hat{X}|\psi\rangle\) is a vector in \(L^2(\mathbb{R})\), since we know from (2) that \(\langle x|\) must map to \(L^2(\mathbb{R})\). Let's take a look at what the general result of \(\langle x|\hat{X}|\psi\rangle\) is: \[\begin{align} \langle x|\hat{X}|\psi\rangle &= \langle x | \hat{X} \int_{-\infty}^{\infty} dx' |x'\rangle\langle x'|\psi\rangle = \int_{-\infty}^{\infty} dx' \langle x | \hat{X} | x'\rangle\langle x'|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx' \langle x | (x'| x'\rangle)\psi(x') \\ &= \int_{-\infty}^{\infty} dx' x'\langle x | x'\rangle\psi(x') = \int_{-\infty}^{\infty} dx' x'\delta(x-x')\psi(x') \\ &= x\psi(x) \end{align}\] If \(\psi(x)=\frac{1}{\sqrt{(1+x^2)\pi}}\), we see that even though it is square-integrable, its result, \(\langle x|\hat{X}|\psi\rangle=x\psi(x)\) is not! \[\begin{align} \int_{-\infty}^{\infty} dx |\psi(x)|^2 &= \frac{1}{\pi}\int_{-\infty}^{\infty} dx \frac{1}{1+x^2}=1 \\ \int_{-\infty}^{\infty} dx |x\psi(x)|^2 &= \frac{1}{\pi}\int_{-\infty}^{\infty} dx \frac{x^2}{1+x^2}=\infty \\ \end{align}\] The implications of this are as we stated above: not all operators have domains over the entire space; some have non-trivial domains. In this case, the vector/wave function \(\psi(x)=\frac{1}{\sqrt{(1+x^2)\pi}}\) is not in the domain of \(\hat{X}\) (though, as you might have noticed, you can restrict the interval over which we integrate, and then get a finite value...).
