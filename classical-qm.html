The Bridge Between Classical and Quantum Mechanics

A Couple Issues We Will Not Address Right Now

Say that we have a point mass moving in one dimension with mass \(m\) and position \(x\):
\(x\in\mathbb{R}\), and since we can measure position, we know there must be an associated observable \(\hat{X}\) which is self-adjoint. Since \(\hat{X}\) is self-adjoint, we know that it has as many eigenvalues as it does possible measured values. \[\hat{X}|x\rangle=x|x\rangle, \sigma(\hat{X})=\mathbb{R}\tag{1}\] This raises two obvious issues:
  1. The spectrum of \(\hat{X}\) is continuous
  2. The spectrum of \(\hat{X}\) is unbounded
This first issue leads us down the following rabbithole: We can no longer say that: \[\sum_x |x\rangle\langle x|=\hat{1}\] because there is no discrete \(x\) to sum over. Instead, we have to say: \[\hat{1}=\int_{-\infty}^{\infty} dx |x\rangle\langle x|\] Clearly, the familiar \(|x\rangle\langle x|\) is no longer a projector, it is a projector density, \(\frac{d\hat{P}}{dt}\). We also run into the following issue: \[\begin{align} |\psi\rangle=\hat{1}|\psi\rangle &= \int_{-\infty}^{\infty} dx |x\rangle\langle x|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx |x\rangle\psi(x) \\ \langle x|\psi\rangle &= \langle x|\hat{1}|\psi\rangle = \langle x|\int_{-\infty}^{\infty} dx' |x'\rangle\langle x'|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx' \langle x|x'\rangle\langle x'|\psi\rangle \\ \end{align}\] (Note that here, we represent \(\langle x|\psi\rangle\) as \(\psi(x)\) which is the position-space wave function.) The only time that each part of this integral is not 0 is when \(x'=x\), so: \[ \langle x|\psi\rangle = \int_{-\infty}^{\infty} dx' \delta(x-x')\langle x'|\psi\rangle \] (Note that this is the Dirac Delta, not the Kroenecker Delta, since the integral of the Dirac Delta is 1. ) This indicates that \(\langle x|x\rangle=\delta(0)=\infty\), so \(|x\rangle\) is not normalizable.

In this situation, \(|x\rangle\) is not a ket that represents the state of quantum system, it is a generalized eigenstate which is used as a mathematical tool only.

The second issue leads us to the following conundrum: The Hellinger-Toeplitz Theorem says that if an operator \(\hat{A}\) is defined on the whole Hilbert space, it is bounded.

As a corollary, if we know in advance that an operator \(\hat{A}]\) is unbounded, we know that it is not defined on the whole Hilbert space.

Some operators have a non-trivial domain.

Take, for example, our position operator \(\hat{X}\): For some state \(|\psi\rangle\), we know that \(\langle\psi|\psi\rangle\lt\infty\). Then: \[ \langle\psi|\hat{1}|\psi\rangle = \langle\psi|\int_{-\infty}^{\infty} dx|x\rangle\langle x|\psi\rangle = \int_{-\infty}^{\infty} dx\langle\psi|x\rangle\langle x|\psi\rangle = \int_{-\infty}^{\infty} dx|\psi(x)|^2 \lt \infty \] Which tells us that \(\psi(x)\in L^2(\mathbb{R})\), or the set of square-integrable functions (functions where \(\int_{-\infty}^{\infty} dx|f(x)|^2 \lt \infty\) is true). This means that we have some functional \(\langle x |\) which has the following action: \[\langle x |:H\to L^2(\mathbb{R}), |\psi\rangle \mapsto \langle x |\psi\rangle\tag{2}\] Based on what we know about linear algebra, we also know that \(\langle x |\psi\rangle\) is the component of the state vector \(|\psi\rangle\) at some position (eigenvalue) \(x\), which corresponds to eigenstate \(|x\rangle\) of \(\hat{X}\) (see definition in (1)).

So, to be able to determine the position of a system in state \(|\psi\rangle\), we first need to project it into the position basis: \(\hat{X}|\psi\rangle\), then check if it exists at a certain position: \(\langle x|\hat{X}|\psi\rangle\).

This will only work if the result of \(\langle x|\hat{X}|\psi\rangle\) is a vector in \(L^2(\mathbb{R})\), since we know from (2) that \(\langle x|\) must map to \(L^2(\mathbb{R})\). Let's take a look at what the general result of \(\langle x|\hat{X}|\psi\rangle\) is: \[\begin{align} \langle x|\hat{X}|\psi\rangle &= \langle x | \hat{X} \int_{-\infty}^{\infty} dx' |x'\rangle\langle x'|\psi\rangle = \int_{-\infty}^{\infty} dx' \langle x | \hat{X} | x'\rangle\langle x'|\psi\rangle \\ &= \int_{-\infty}^{\infty} dx' \langle x | (x'| x'\rangle)\psi(x') \\ &= \int_{-\infty}^{\infty} dx' x'\langle x | x'\rangle\psi(x') = \int_{-\infty}^{\infty} dx' x'\delta(x-x')\psi(x') \\ &= x\psi(x) \end{align}\] If \(\psi(x)=\frac{1}{\sqrt{(1+x^2)\pi}}\), we see that even though it is square-integrable, its result, \(\langle x|\hat{X}|\psi\rangle=x\psi(x)\) is not! \[\begin{align} \int_{-\infty}^{\infty} dx |\psi(x)|^2 &= \frac{1}{\pi}\int_{-\infty}^{\infty} dx \frac{1}{1+x^2}=1 \\ \int_{-\infty}^{\infty} dx |x\psi(x)|^2 &= \frac{1}{\pi}\int_{-\infty}^{\infty} dx \frac{x^2}{1+x^2}=\infty \\ \end{align}\] The implications of this are as we stated above: not all operators have domains over the entire space; some have non-trivial domains. In this case, the vector/wave function \(\psi(x)=\frac{1}{\sqrt{(1+x^2)\pi}}\) is not in the domain of \(\hat{X}\) (though, as you might have noticed, you can restrict the interval over which we integrate, and then get a finite value...).

Bringing classical canonical transformations to quantum mechanics

So far in our quantum mechanics toolkit, we have the observables \(\hat{x}\) and \(\hat{p}\) that allow us to measure position and momentum, and we have briefly discussed the family of translation operators, \(\{\hat{T}_{\Delta x}\}\). How do we bring in the classical canonical transformations to quantum mechanics? By Stone's Theorem we know that \[\hat{T}_{\Delta x}=e^{-i\Delta x \hat{O}}, \hat{O}=\hat{O}^\dagger\] If we only consider infinitesimally small translations, we can now apply the Taylor expansion for \(e^A\) and discard any terms higher than order \(\Delta x\), because \(\Delta x^2\) would be teeny tiny: \[\begin{align} e^A &= \hat{1}+A+\frac{A^2}{2!}+\frac{A^3}{3!}+\cdots \\ \hat{T}_{\Delta x} &= \hat{1} - i\Delta x\hat{O}+o(\Delta x^2) \tag{3} \end{align}\] Classically we know that momentum is the infinitesimal generator of space translation. We want this to still apply now, when we work in quantum mechanics, and say that \(\hat{O}\) is proportional to \(\hat{p}\). However, we see that each term in (3) must be dimensionless, and \(\Delta x\) has units of length. So we say that \(\hat{O}=\frac{\hat{p}}{\beta}\), where \(\beta\) has units of action, or units of length times the units of momentum: \([L][L\cdot M\cdot T^{-1}]\) Experimentally, they've found \(\beta\) to be \(\hbar\).

In fact, for any (classical) canonical unitary transformation \(U_{d\alpha}\) generated by some \(G\), there is a (quantum) operator \(\hat{U}_{d\alpha}=\hat{1}-id\alpha\frac{\hat{G}}{\hbar} + o(d\alpha^2)\). We can apply this to the classical momentum boosts and time translations. To summarize:

Generically, a canonical classical transformation \(U_{d\alpha}\) generated by some \(G\) has a quantized counterpart generated by \(\hat G\): \[\hat{U}_{d\alpha}=\hat{1}-id\alpha\frac{\hat{G}}{\hbar} + o(d\alpha^2)\] \[\hat U_{d\alpha}|\psi(\alpha)\rangle = |\psi(\alpha+d\alpha\rangle)\]
Time Translations are generated by \(\hat H\) and act as: \[\hat{U}_{\Delta t} = e^{- i\Delta t\frac{\hat{H}}{\hbar}} = \hat{1} - i\Delta t\frac{\hat{H}}{\hbar} + o(\Delta t^2) \] \[\hat{U}_{\Delta t} |\psi(t)\rangle = |\psi(t + \Delta t)\rangle\] Space Translations are generated by \(\hat p\) and act as: \[\hat{T}_{\Delta x} = e^(i\Delta x\frac{\hat{p}}{\hbar}) = \hat{1} - i\Delta x\frac{\hat{p}}{\hbar} + o(\Delta x^2) \] \[\hat{T}_{\Delta x} |\psi(x)\rangle = |\psi(x + \Delta x)\rangle\] Momentum Boosts are generated by \(-\hat x\) and act as: \[\hat{B}_{\Delta p} = e^{i\Delta p\frac{\hat{x}}{\hbar}} = \hat{1} + i\Delta p\frac{\hat{x}}{\hbar} + o(\Delta p^2) \] \[\hat{B}_{\Delta p} |\psi(p)\rangle = |\psi(p + \Delta p)\rangle\]


Now that we know \[\hat{T}_{\Delta x} = \hat{1} - i\Delta x\frac{\hat{p}}{\hbar}+o(\Delta x^2) \tag{4}\] we can use \(\hat{T}_{\Delta x}\) to connect the canonical Poisson Bracket to the quantum mechanics commutator.

The Canonical Poisson Bracket and the Commutator

To draw this connection, we first demonstrate.a few things about \(\hat x\) and \(\hat p\). The Commutator \([\hat{x},\hat{p}]\). The demonstration is as follows: \[\begin{align} \hat{T}_{\Delta x}\hat{x}\hat{T}_{-\Delta x}|x\rangle &= \hat{T}_{\Delta x}\hat{x}|x-\Delta x\rangle \\ &= \hat{T}_{\Delta x} (x-\Delta x)|x-\Delta x\rangle \\ &= (x-\Delta x)\hat{T}_{\Delta x} |x-\Delta x\rangle \\ &= (x-\Delta x)|x-\Delta x+\Delta x\rangle \\ &= (x-\Delta x)|x\rangle = x|x\rangle - \Delta x|x\rangle = \hat{x}|x\rangle - \Delta x|x\rangle\\ &= (\hat{x}-\Delta x)|x\rangle \end{align}\] From which we can gather that \(\hat{T}_{\Delta x}\hat{x}\hat{T}_{-\Delta x} = \hat{x}-\Delta x\). Using the expansion of \(e^A\) mentioned above again, we can write: \[\begin{align} \hat{T}_{\Delta x}\hat{x}\hat{T}_{-\Delta x} &= \hat{x}-\Delta x\\ &= (\hat{1} - i\Delta x\frac{\hat{p}}{\hbar}+o(\Delta x^2))\hat{x}(\hat{1} + i\Delta x\frac{\hat{p}}{\hbar}+o(\Delta x^2)) \\ \hat{x}-\Delta x&= \hat{x} + i\Delta x/\hbar\hat{x}\hat{p}-i\Delta x/\hbar\hat{p}\hat{x} + o(\Delta x^2) \\ \end{align}\] In this last line, we see that there is no \(\Delta x^2\) term on the left hand side, so the \(o(\Delta x^2)\) on the right hand side must equal 0. \[\begin{align} \hat{x}-\Delta x &= \hat{x} + i\Delta x/\hbar\hat{x}\hat{p}-i\Delta x/\hbar\hat{p}\hat{x} + o(\Delta x^2) \\ \hat{x}-\Delta x &= \hat{x} + i\Delta x/\hbar\hat{x}\hat{p}-i\Delta x/\hbar\hat{p}\hat{x} \\ -\Delta x &= i\Delta x/\hbar\hat{x}\hat{p}-i\Delta x/\hbar\hat{p}\hat{x} \\ -1 &= i/\hbar\hat{x}\hat{p}-i/\hbar\hat{p}\hat{x} \\ &= i/\hbar(\hat{x}\hat{p}-\hat{p}\hat{x}) \\ -1 &= i/\hbar[\hat{x},\hat{p}] \\ -\hbar/i &= [\hat{x},\hat{p}] \\ i\hbar &= [\hat{x},\hat{p}] \end{align}\] \(\hat{p}\) in Position-space. Say that we have position-space wave function \(\psi(x)=\langle x|\psi\rangle\). Performing \(\langle x|\hat{T}_{\Delta x}|\psi\rangle\) shifts the wave function by \(\Delta x\) and then evaluates at a specific value of \(x\). \[\begin{align} \langle x|\hat{T}_{\Delta x}|\psi\rangle &= \langle \hat{T}^{\dagger}_{\Delta x} x|\psi\rangle = \langle \hat{T}^{-1}_{\Delta x} x|\psi\rangle = \langle \hat{T}_{-\Delta x} x|\psi\rangle \\ &= \langle x-\Delta x|\psi\rangle = \psi(x-\Delta x)\\ &= \langle x|\psi\rangle-\Delta x\frac{d}{dx}\langle x|\psi\rangle + o(\Delta x^2) \end{align}\] where the last line follows from the first-order Taylor expansion that we used here: \(f(x+dx) = f(x)+dxg(x)+o(dx^2)\). We can take an alternative route in evaluating \(\langle x|\hat{T}_{\Delta x}|\psi\rangle\) by using (4): \[\begin{align} \langle x|\hat{T}_{\Delta x}|\psi\rangle &= \langle x|\hat{1} - i\Delta x\frac{\hat{p}}{\hbar}+o(\Delta x^2)|\psi\rangle \\ &= \langle x|\psi\rangle - i\Delta x/\hbar\langle x|\hat{p}|\psi\rangle + o(\Delta x^2) \end{align}\] We can equate our two results: \[\begin{align} \langle x|\psi\rangle-\Delta x\frac{d}{dx}\langle x|\psi\rangle + o(\Delta x^2) &= \langle x|\psi\rangle - i\Delta x/\hbar\langle x|\hat{p}|\psi\rangle + o(\Delta x^2) \\ \frac{d}{dx}\langle x|\psi\rangle &= i/\hbar\langle x|\hat{p}|\psi\rangle \\ -i\hbar\frac{d}{dx}\langle x|\psi\rangle &= \langle x|\hat{p}|\psi\rangle \\ -i\hbar\frac{d}{dx}\langle x| &= \langle x|\hat{p} \end{align}\] The commutator \([\hat p_i, \hat{p}_j]\) is similar to \(\{p_i, p_j\}_{P.B.}\). We can expand to more than one dimension, so that \(\vec{r}\) and \(\vec{p}\) are used to represent the position and momentum in all three directions. If we consider a translation in three dimensions: \[\begin{cases} \vec{r} \to \vec{r}' = \vec{r}+\vec{a} \\ \vec{p} \to \vec{p} \end{cases}\] where \(\vec{a}\) is a vector with three elements that describes the change in all three dimensions. The generator for these (canonical) transformations is \(\vec{a}\cdot\vec{p}\). We can create a one-paramater group of these transformations if we restrict the direction of \(\vec{a}\) and only vary its magnitude. Our family of transformations looks like: \[\hat{T}_{\vec{a}}=e^{-i\vec{a}\cdot\vec{p}/\hbar}\] We know (or want to enforce) that this group of transformations is an abelian group, so \(\hat{T}_\vec{a}\hat{T}_\vec{b}=\hat{T}_\vec{b}\hat{T}_\vec{a}\) for two different vectors \(\vec{a}=a\cdot\hat{e}_i, \vec{b}=b\cdot\hat{e}_i\) where \(\hat{e}_i\) is some unit vector. \[\begin{align} \hat{T}_\vec{a}\hat{T}_\vec{b} &= (\hat{1} - ia\hat{p}_i/\hbar+o(a^2))(\hat{1} - ib\hat{p}_j/\hbar+o(b^2)) \\ &= (\hat{1} - ib\hat{p}_i/\hbar+o(b^2))(\hat{1} - ia\hat{p}_j/\hbar+o(a^2)) = \hat{T}_\vec{b}\hat{T}_\vec{a} \\ &\hat{1}-ia\hat{p}_i/\hbar-ib\hat{p}_j/\hbar-ab\hat{p}_i\hat{p}_j/\hbar^2 + o(a^2)+o(b^2) \\ &=\hat{1}-ib\hat{p}_j/\hbar-ia\hat{p}_i/\hbar-ab\hat{p}_j\hat{p}_i/\hbar^2 + o(a^2)+o(b^2) \\ -ab\hat{p}_i\hat{p}_j/\hbar^2 &= -ab\hat{p}_j\hat{p}_i/\hbar^2 \\ \hat{p}_i\hat{p}_j &= \hat{p}_j\hat{p}_i \end{align}\] which of course tells us that \([\hat{p}_i,\hat{p}_j]=0\). This mirrors the result that \(\{p_i, p_j\}_{P.B.}=0\). We can also do the same thing with momentum boost and \(\hat{B}_\vec{a}\) to find that \([\hat{x}_i, \hat{x}_j]=0\).

The commutator \([\hat{x}_i,\hat{p}_j]\) is similar to \(\{q_i, p_j\}_{P.B.}\). Knowing that \(\langle x|\hat{p} = -i\hbar\frac{d}{dx}\langle x|\), we can do: \[\begin{align} \langle\vec{r}|[\hat{x}_i,\hat{p}_j] &= \langle\vec{r}|\hat{x}_i\hat{p}_j - \langle\vec{r}|\hat{p}_j\hat{x}_i \\ &= x_i\langle\vec{r}|\hat{p}_j+i\hbar\frac{d}{dx_j}\langle\vec{r}|\hat{x_i} \\ &= x_i(-i\hbar\frac{d}{dx_j})\langle\vec{r}| + i\hbar\frac{d}{dx_j}x_i\langle\vec{r}| \\ &= -ix_i\hbar\frac{d}{dx_j}\langle\vec{r}| + i\hbar(x_i\frac{d}{dx_j}\langle\vec{r}| + \langle\vec{r}|\frac{d}{dx_j}x_i) \\ &= -ix_i\hbar\frac{d}{dx_j}\langle\vec{r}| + i\hbar(x_i\frac{d}{dx_j}\langle\vec{r}| + \langle\vec{r}|\delta_{ij}) \\ &= i\hbar\langle\vec{r}|\delta_{ij} \end{align}\] which tells us that \([\hat{x}_i,\hat{p}_j] = i\hbar\delta_{ij}\). This mirrors the result that \(\{q_i, p_j\}_{P.B.}=\delta_{ij}\)

The connection between \(\{G,A\}_{PB}\) and \([\hat G, \hat A]\). Classically, we know the generic infinitesimal canonical transformation: \[\begin{cases} q \to q'=q+d\alpha\frac{\partial G}{\partial p} \\ p \to p'=p-d\alpha\frac{\partial G}{\partial q} \\ \end{cases}\] and that phase space function \(A\mapsto A'=A+d\alpha\{A,G\}_{PB}\) (see here). Call this unitary transformation \(\hat{U}_{d\alpha}\) so \(\hat{U}_{d\alpha}|\psi\rangle = |\psi'\rangle\). The expectation value of \(\hat{A}\) will change as \(|\psi\rangle\) changes under \(\hat{U}_{d\alpha}\), and we know from earlier that \(\hat{U}_{d\alpha}=\hat{1}-id\alpha\frac{\hat{G}}{\hbar}\). \[\begin{align} \langle\hat{A}\rangle_\psi \mapsto \langle\hat{A}\rangle_{\psi'} &= \langle\hat{U}_{d\alpha}\psi|\hat{A}|\hat{U}_{d\alpha}\psi\rangle \\ &= \langle\psi|\hat{U}^\dagger_{d\alpha}\hat{A}\hat{U}_{d\alpha}|\psi\rangle \\ &= \langle\psi|(\hat{1}+id\alpha\frac{\hat{G}}{\hbar})\hat{A}(\hat{1}-id\alpha\frac{\hat{G}}{\hbar})|\psi\rangle \\ &= \langle\psi|\hat{A}-\frac{id\alpha}{\hbar}\hat{A}\hat{G} + \frac{id\alpha}{\hbar}\hat{G}\hat{A}-o(d\alpha^2)|\psi\rangle \\ &= \langle\hat{A}\rangle_\psi + \frac{id\alpha}{\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle + o(d\alpha^2) \end{align}\] Compare this to \(A'=A+d\alpha\{A,G\}_{PB}\). This is just the classical function \(A\), not the operator "version" \(\hat{A}\), but the parallel is somewhat apparent; \(d\alpha\{A,G\}_{PB}\) corresponds to \(\frac{id\alpha}{\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle\), indicating that the "operator version" of \(\{G,A\}_{PB}\) is: \[\begin{align} d\alpha\{A,G\}_{PB} &= \frac{id\alpha}{\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle \\ \{A,G\}_{PB} = -\{G,A\}_{PB} &= \frac{i}{\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle \\ \{G,A\}_{PB} &= -\frac{i}{\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle \\ &= \frac{1}{i\hbar}\langle\psi|[\hat{G}, \hat{A}]|\psi\rangle = \langle\psi|\frac{[\hat{G}, \hat{A}]}{i\hbar}|\psi\rangle \\ \end{align}\] or that \({\{\widehat{G,A}\}_{PB}} = \frac{[\hat{G}, \hat{A}]}{i\hbar}\).
