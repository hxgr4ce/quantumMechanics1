The Completeness of \(\mathbb{R}\) and \(\mathbb{Q}\)

Metric incompleteness

A metric space is a set \(S\) equipped with the concept of distance between its elements (e.g. points), where the distance \(d\) has the following properties: \[d: S \times S \to \mathbb{R}\]

  • The distance between a point and itself is 0: \(\forall a \in S, d(a, a) = 0\)
  • Positivity: \(\forall a, b, a \neq b \in S, d(a, b)>0\)
  • Symmetry: \(\forall a, b \in S, d(a, b)=d(b, a)\)
  • Triangle inequality: \(\forall a, b, c \in S, d(x, z) \leq d(x,y)+d(y, z)\)

A Cauchy Sequence is a sequence with elements that converge so they are eventually arbitrarily close to each other. Or:

A sequence \((x_n)_{n=1}^{\infty}\) is a Cauchy sequence if for all small positive distances \(\epsilon\), \(\exists N \in \mathbb{N} \mid \forall i,j \geq N, d(x_i,x_j) \lt \epsilon\) (All points indexed by all pairs of \(i\) and \(j \geq N\) are closer together than distance \(\epsilon\)).

There are infinitely many Cauchy sequences in a non-trivial metric space. If all Cauchy sequences do not converge in a field, it implies that the sequence is attempting to converge on a value that does not exist in the field.

(A number of theorems require this type of completeness, e.g, Intermediate Value Theorem, Extreme Value Theorem for \(\mathbb{R}\), Banach fixed-point theorem, and infinite series analysis.)

\(\mathbb{Q}\) is not metrically complete. For example, the sequence \((\frac{F_{i-1}}{F_i})_{i=1}^{\infty}\) where, \(F_i\) is the \(i\)th term in the Fibonacci sequence, converges on the irrational value \(\phi\), which is clearly not an element of the rational numbers \(\mathbb{Q}\).

\(\mathbb{R}\) fixes these issues; in fact, can be generated from \(\mathbb{Q}\) using the set of all Cauchy sequences, though that's outside the scope of this class, I'm pretty sure.

Algebraic incompleteness

Though \(\mathbb{R}\) is metrically complete where \(\mathbb{Q}\) is metrically incomplete, both of these fields are still algebraically incomplete/closed.

Given a polynomial in field \(F\): \[P_m(x)= \sum_{i=0}^m c_ix^m, \forall c_i, x \in F\] \(F\) is only algebraically closed if \(\forall P_m, \exists \{\lambda_i\}_{i=1,...,m}, \lambda_i \in F \mid P_m(x) = \Pi_{i=1}^m (x-\lambda_i)\)

Essentially, a field is only algebraically closed if the solutions to each possible polynomials is an element of the field.

Since the polynomial \(x^2-2=0\) only has irrational solutions \(\pm\sqrt2\), \(\mathbb{Q}\) is not algebraically complete, and since the polynomials \(x^2 + 1 = 0\) only has imaginary solution \(\pm i\), \(\mathbb{R}\) is not algebraically closed, either.

\(\mathbb{C}\) is both metrically and algebraically closed, so we often use it in quantum mechanics.
